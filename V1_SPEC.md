# PagePal - V1 Specification

**Goal**: Chrome extension that lets users customize reading experience (font, spacing, contrast, clutter) on any webpage via a popup.

## Core Features for V1

1. **Toggle dyslexia-friendly font** - Switch to OpenDyslexic or similar readable font
2. **Adjustable font size + line spacing + word spacing** - Fine-tune text readability
3. **High-contrast themes** - Light / Dark / High Contrast options
4. **Distraction-free mode** - Hide sidebars, ads, popups
5. **Line highlighting** - Highlight the line/block the user hovers over

## Target Users
People with dyslexia, visual impairments, or anyone who wants better control over web readability.
